
# OpenCV를 통한 관점 변환
이 프로젝트는 OpenCV를 사용하여 이미지에서 사용자가 마우스로 클릭한 4개의 지점을 기준으로 **원근 변환(Perspective Transform)**을 수행하는 파이썬 도구입니다. 찌그러진 문서나 간판 사진을 정면에서 본 것처럼 평평하게 펼칠 때 유용합니다.

## 주요 기능
* 마우스 인터랙티브:이미지 위를 직접 클리하여 좌표를 지정합니다.
* 실시간 피드백: 클릭한 지점에 점을 표시하여 위치를 확인합니다.
* 자동 변환: 4개의 점이 모두 선택되면 즉시 변환된 결과를 새 창에 띄웁니다.

## 사용 방법
1. 코드 실행후 original 이미지의 4곳을 클릭합니다.
2. 클릭 직후, new_image 창에 평면으로 펼쳐진 결과가 나타납니다.
